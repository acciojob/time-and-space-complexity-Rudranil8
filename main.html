Time Complexity and Space Complexity

Time complexity of an algorithm quantifies the amount of time taken by an algorithm to run as a function of the length of the input.The valid algorithm takes a finite amount of time for execution. The time required by the algorithm to solve given problem is called time complexity of the algorithm. Efficient keyword.

The Space Complexity of an algorithm is the amount of memory required by the algorithm to solve a given problem. It is typically expressed as O(n), where n is the size of the input.
